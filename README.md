# 💗 PINK CITY MERGE

同じ「ハート」をくっつけて、どんどん大きく進化させていくスイカゲーム風のマージパズル。
育てきると最終形態 **PINK CITY** 👑 が完成する、ネオン×ホログラフィックな1ファイル完結のWebゲームです。

---

## 🎮 遊び方

1. **「はじめる ✨」** ボタンを押してスタート
2. 上から落ちてくるハートを、**指（スマホ）** または **マウス（PC）** で左右に動かす
3. 指やマウスを **離す（リリース）** とハートが落下
4. **同じ種類のハートどうしがぶつかると合体**して、1つ上のランクに進化！
5. 上の **点線（デッドライン）** を超えてハートが積み上がるとゲームオーバー

### ⌨️ キーボード操作（PC）

| キー | 動作 |
|------|------|
| ← / → | ハートを左右に移動 |
| スペース | ハートを落とす |
